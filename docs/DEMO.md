# 功能演示说明

## 🎬 使用流程演示

### 1. 启动应用

双击 `start.bat`（Windows）或运行 `npm start`

```

========================================

✅ 依赖已存在

[3/3] 启动服务器...

🚀 服务即将启动...
📱 访问地址: http://localhost:3000
```

### 2. 打开浏览器

访问 http://localhost:3000

页面显示：
- 🔥 GitHub 贡献热力图生成器（标题）
- 输入框：输入 GitHub 用户名
- 颜色主题选择器
- "生成热力图"按钮

### 3. 输入用户名

在输入框中输入：`octocat`

选择颜色主题：默认绿色

点击"生成热力图"按钮

### 5. 下载图片

点击任意图表下方的"下载"按钮：
- 📥 下载热力图 → 保存为 `heatmap.svg`
- 📥 下载语言图 → 保存为 `languages.svg`
- 📥 下载分享卡片 → 保存为 `share-card.svg`

### 6. 切换颜色主题

在顶部的颜色主题下拉框中选择：
- 默认绿色
- 蓝色
- 紫色
- 橙色

重新点击"生成热力图"即可看到不同主题的效果图。

## 🌟 特色功能展示

### 1. 响应式设计
- 桌面端：完整布局，多列显示
- 平板端：自适应调整
- 手机端：单列堆叠，易于浏览

### 2. 加载状态
点击生成后显示：
```
        ⟳
  正在获取数据...
```

### 3. 错误提示
如果用户不存在：
```
┌─────────────────────────────────────┐
│ ⚠️ 用户不存在                        │
└─────────────────────────────────────┘
```

### 4. 键盘快捷操作
- 输入框中按 Enter 键 = 点击生成按钮
- 页面加载后自动聚焦输入框

## 💡 使用技巧

### 技巧 1: 批量生成
可以快速切换用户名连续生成多个用户的图表

### 技巧 2: 主题对比
同一用户用不同主题生成，对比效果

### 技巧 3: SVG 编辑
下载的 SVG 文件可以用矢量编辑软件（如 Illustrator、Inkscape）进一步编辑

### 技巧 4: 嵌入网页
SVG 可以直接嵌入到个人网站或博客中

### 技巧 5: 打印友好
SVG 格式打印质量高，适合制作简历附件

## 🎯 典型应用场景

### 场景 1: 求职简历
1. 生成自己的贡献热力图
2. 下载 SVG 格式
3. 转换为 PNG 插入简历
4. 展示编程活跃度

### 场景 2: 技术分享
1. 生成团队成员的图表
2. 对比不同成员的技术栈
3. 在 PPT 中展示分析结果

### 场景 3: 社交媒体
1. 生成精美的分享卡片
2. 下载到本地
3. 发布到 Twitter/LinkedIn/微信朋友圈

### 场景 4: 开源项目推广
1. 生成项目维护者的贡献图
2. 展示项目的活跃度
3. 吸引更多贡献者
