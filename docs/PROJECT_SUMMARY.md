|------|------|------|----------|

## 🎯 技术亮点

### 1. 无原生依赖
- 使用 SVG 替代 Canvas，无需编译原生模块
- 跨平台兼容性好（Windows/Mac/Linux）
- 安装简单，无需构建工具

### 2. 智能降级
- GraphQL API 优先（数据更详细）
- REST API 降级（兼容性更好）
- 自动选择最佳数据源

### 3. 安全性
- XML 转义防止 XSS 攻击
- HTML 转义防止注入
- 输入验证和错误处理

### 4. 性能优化
- 并行 API 请求（Promise.all）
- SVG 轻量级输出（比 PNG 小）
- 前端懒加载图片

### 5. 用户体验
- 实时加载状态反馈
- 清晰的错误提示
- 一键下载功能
- 响应式设计

## 🚀 部署建议

### 本地开发
```bash

```

### 生产环境
1. 配置 `.env` 文件（添加 GITHUB_TOKEN）
2. 设置环境变量
3. 使用 PM2 管理进程
4. 配置 Nginx 反向代理
5. 启用 HTTPS

可以创建 Dockerfile 进行容器化部署。
